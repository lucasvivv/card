## Digital Business Card

This project is a digital business card website for my Web Typography class. The goal was to create a simple webpage that shows my contact information and portfolio in a clean and visually styled way. I built it using HTML and CSS and structured the layout to resemble the mockup of my business card.

## Process

My process started with making mockups in Figma where I tested different layouts, typefaces, and colors. After getting feedback from classmates on the strongest option, I moved on to building the site with HTML and CSS.

## What I Learned

I learned more about structuring HTML, linking fonts, styling layouts with CSS, and working with images and graphics on a webpage. I also got more comfortable organizing files like assets, images, and stylesheets.

## Personal Branding

My personal branding is something I’ve been developing over time as a graphic design student. This project helped me apply that branding to the web, including my typography choices and personal logo/graphic.

## Fonts

I used IM Fell English SC from Google Fonts. I chose it because it has a classic, editorial style that fits the visual identity I’m building for my brand.
